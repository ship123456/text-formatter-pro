## Overview

Text Formatter Pro is a browser extension that provides text formatting, cleanup, and analysis utilities.

The extension is designed to process user input directly inside the browser without requiring accounts or external services.

## Core Privacy Principles

Text Formatter Pro follows these principles:

- Keep processing inside the browser whenever possible
- Request only required extension permissions
- Allow users to manage stored information
- Avoid unnecessary data collection

## Zero Data Collection Policy

Text Formatter Pro does not collect personal information through its extension features.

The extension does not require:

- User registration
- Login credentials
- Personal profile information

No analytics or user tracking features are included.

## Sensitive Data

Text Formatter Pro does not require users to provide sensitive information such as:

- Personal identity information
- Account credentials
- Payment information

Users control the text they choose to enter into the extension.

## Local-Only Text Processing

Text operations are handled locally inside the browser.

Examples:

- Uppercase conversion
- Lowercase conversion
- Text cleanup
- Duplicate line removal
- Sorting lines
- Find and replace
- Text statistics calculation

Text formatting features do not require sending content to external services.

## Local Storage Usage

Text Formatter Pro uses Chrome Storage API to store application settings.

Stored information may include:

- Character limit configuration
- Auto Save settings
- Recent text entries
- Text in text editor

## Auto Save & Text History Storage

When enabled, Auto Save stores current text locally.

Recent Text History allows users to restore previous text entries.

Users can:

- Disable Auto Save
- Delete individual history items
- Clear all stored history

## Data Retention

Stored data remains available until:

- User clears extension data
- User removes saved history
- Browser storage is cleared
- Extension data is reset

## Clipboard Access

Clipboard actions occur only when selected by the user.

Clipboard features include:

- Copying formatted text
- Pasting text into the editor

Clipboard content is accessed only when the user manually performs copy or paste actions.

## Extension Permissions Explanation

Text Formatter Pro uses limited permissions.

### Storage Permission

Required for:

- Saving character limit
- Saving Text of text editor 
- Maintaining recent history
- Supporting Auto Save functionality

The extension requests only permissions required for its features.

## Data Sharing Policy

Text Formatter Pro does not:

- Sell user information
- Share user data with third parties
- Transfer text content for external processing

## Network Access Policy

Text Formatter Pro does not rely on external servers for text formatting features.

Core functionality works through browser-based processing.

## Third-Party Services

Text Formatter Pro does not use third-party services for:

- Text processing
- User tracking
- Data analytics

## Managing & Clearing Stored Data

Users can manage stored information by:

1. Opening Text Formatter Pro.
2. Accessing menu options where we have        
   recent history also Clear All option down below the application 
3. Clearing history or resetting preferences.

Users can also remove extension data through browser settings.

## User Data Control

Users can also remove stored extension data by clearing the extension's browser storage or removing the extension from the browser.

Available controls:

- Enable or disable Auto Save
- Manage recent text history
- Change preferences
- Reset application settings

## Privacy Limitations

Text Formatter Pro storage depends on browser storage behavior.

Stored information may be removed if:

- Browser data is cleared
- Extension storage is deleted
- Extension is removed

## Policy Updates

This privacy document may be updated if:

- New features are added
- Storage behavior changes
- Permission usage changes
